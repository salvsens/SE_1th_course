Сформулировать теорему о структуре общего решения неоднородной системы линейных алгебраических уравнений и доказать её.	Теорема. Пусть известно частное решение \(x^*\) СЛАУ \(Ax = b\). Тогда любое решение этой СЛАУ может быть представлено в виде: \(x = x^* + c_1\xi_1 + \cdots + c_k\xi_k\), где \(c_1,...,c_k\) – некоторые постоянные, а \(\xi_1,...,\xi_k\) – ФСР соответствующей однородной системы \(Ax = 0\). Доказательство. \(X_{\text{общ. неодн.}} = X_{\text{частн. неодн.}} + X_{\text{общ. одн.}}\) <br> Пусть \(x^0\) – произвольное решение СЛАУ \(Ax = b \Rightarrow x^0 - x^*\) – решение СЛАУ \(Ax = 0\) (по свойствам решений СЛАУ). К \(x^0 - x^*\) применим теорему о структуре общего решения однородной СЛАУ: \(x^0 - x^* = c_1\xi_1 + \cdots + c_k\xi_k \Rightarrow x^0 = x^* + c_1\xi_1 + \cdots + c_k\xi_k\).
Вывести формулу Муавра и доказать её.	Утверждение. Формула Муавра: \(z^n = r^n(\cos n\varphi + i\sin n\varphi)\), \(n \in \mathbb{N}\). Доказательство. Применим принцип математической индукции. <br> 1) \(n = 2\): \(z^2 = z \cdot z = r \cdot r \cdot (\cos(\varphi+\varphi) + i\sin(\varphi+\varphi)) = r^2(\cos 2\varphi + i\sin 2\varphi)\). <br> 2) Предположим, что формула верна для всех \(n \leq k\). Покажем, что из этого следует, что она верна для всех \(n = k+1\): \(z^{k+1} = z^k \cdot z = r^k(\cos k\varphi + i\sin k\varphi) \cdot r(\cos\varphi + i\sin\varphi) = r^{k+1}(\cos(k\varphi+\varphi) + i\sin(k\varphi+\varphi)) = r^{k+1}(\cos((k+1)\varphi) + i\sin((k+1)\varphi))\). Таким образом, формула верна \(\forall n \in \mathbb{N}\).
Доказать, что если у многочлена с действительными коэффициентами есть корень в ненулевой мнимой части, то число, комплексно сопряжённое к этому корню, тоже будет корнем этого многочлена той же кратности.	Утверждение. Если \(c \in \mathbb{C}\) – корень кратности k многочлена \(P_n(x)\) с действительными коэффициентами, то \(\bar{c}\) тоже является корнем \(P_n(x)\) кратности k. Доказательство. Пусть \(P_n(c) = a_n \cdot c^n + \cdots + a_1 \cdot c + a_0 = 0\). Сопрягая обе части: \(\overline{0} = \overline{a_n \cdot c^n + \cdots + a_1 \cdot c + a_0}\). Откуда \(\bar{c}\) тоже будет корнем: \(0 = a_n \cdot \bar{c}^n + \cdots + a_1 \cdot \bar{c} + a_0 = a_n \cdot c^n + \cdots + a_1 \cdot c + a_0\). Если \(c\) – корень кратности 1, то всё доказано. Если кратность \(> 1\), то делим на \(x - \bar{c}\), по теореме Безу остаток будет нулевым и к многочлену применяем ту же процедуру.
Вывести формулу для вычисления векторного произведения в правом ортонормированном базисе трёхмерного пространства и привести её вывод.	Утверждение. Пусть \(\vec{i}, \vec{j}, \vec{k}\) – правый ОНБ, \(\vec{a} = a_x\vec{i} + a_y\vec{j} + a_z\vec{k}\), \(\vec{b} = b_x\vec{i} + b_y\vec{j} + b_z\vec{k}\). Тогда: \([\vec{a},\vec{b}] = \begin{vmatrix}\vec{i}&\vec{j}&\vec{k}\\a_x&a_y&a_z\\b_x&b_y&b_z\end{vmatrix} = \vec{i}(a_yb_z - a_zb_y) + \vec{j}(a_zb_x - a_xb_z) + \vec{k}(a_xb_y - a_ybx)\). Доказательство. Т.к. \(\vec{i}, \vec{j}, \vec{k}\) – ОНБ, то \(\vec{i}\times\vec{i} = \vec{j}\times\vec{j} = \vec{k}\times\vec{k} = \vec{0}\), \(\vec{i}\times\vec{j} = \vec{k}\), \(\vec{j}\times\vec{i} = -\vec{k}\), \(\vec{i}\times\vec{k} = -\vec{j}\), \(\vec{k}\times\vec{i} = \vec{j}\), \(\vec{j}\times\vec{k} = \vec{i}\), \(\vec{k}\times\vec{j} = -\vec{i}\). Тогда: \(\vec{a}\times\vec{b} = a_xb_y(\vec{i}\times\vec{j}) + a_xb_z(\vec{i}\times\vec{k}) + a_yb_x(\vec{j}\times\vec{i}) + a_yb_z(\vec{j}\times\vec{k}) + a_zb_x(\vec{k}\times\vec{i}) + a_zb_y(\vec{k}\times\vec{j}) = \vec{i}(a_yb_z - b_ya_z) + \vec{j}(a_zb_x - a_xb_z) + \vec{k}(a_xb_y - a_ybx) = \begin{vmatrix}\vec{i}&\vec{j}&\vec{k}\\a_x&a_y&a_z\\b_x&b_y&b_z\end{vmatrix}\).
Доказать теорему о том, что любое линейное уравнение на координаты точки в трёхмерном пространстве задаёт плоскость и что любая плоскость определяется линейным уравнением.	Теорема. 1) Любая плоскость в пространстве определяется уравнением \(Ax + By + Cz + D = 0\), в котором \(A, B, C, D\) – некоторые числа. 2) Любое уравнение \(Ax + By + Cz + D = 0\), где \(A^2 + B^2 + C^2 > 0\), определяет в пространстве плоскость. Доказательство. <br> 1) Рассмотрим плоскость \(\pi\). Пусть точка \(M_0(x_0, y_0, z_0)\) ей принадлежит. Рассмотрим \(\vec{n} \perp \pi\). Пусть \(\vec{n} = (A, B, C)\). \(M(x,y,z) \in \pi \Leftrightarrow (\vec{n}, \overrightarrow{M_0M}) = 0 \Leftrightarrow A(x_0-x) + B(y_0-y) + C(z_0-z) = 0\). Т.е. \(Ax + By + Cz + D = 0\), где \(D = -Ax_0 - By_0 - Cz_0\). Таким образом, координаты точки M удовлетворяют уравнению \(Ax + By + Cz + D = 0\). <br> 2) Рассмотрим уравнение \(Ax + By + Cz + D = 0\), где \(A^2+B^2+C^2 > 0\). Оно имеет хотя бы одно решение (например, если \(A \neq 0\), то \(x_0 = -\frac{D}{A}, y_0 = z_0 = 0\)). Обозначим за \(M_0\) точку \((x_0, y_0, z_0)\). Пусть точка \(M(x,y,z)\) удовлетворяет уравнению. Вычтем из него равенство \(Ax_0 + By_0 + Cz_0 + D = 0\): \(A(x_0-x) + B(y_0-y) + C(z_0-z) = 0 \Leftrightarrow (\vec{n}, \overrightarrow{M_0M}) = 0\), где \(\vec{n} = (A,B,C)\). \((\vec{n}, \overrightarrow{M_0M}) = 0 \Leftrightarrow \vec{n} \perp \overrightarrow{M_0M} \Leftrightarrow\) точка M лежит в плоскости, проходящей через \(M_0\) и перпендикулярной вектору \(\vec{n} \Rightarrow\) уравнение \(Ax + By + Cz + D = 0\) определяет плоскость.
Дать определение фундаментальной системы решений (ФСР) однородной системы линейных уравнений. Доказать теорему о существовании ФСР.	Определение. Любые \(n - r\) линейно независимых столбцов, являющихся решениями однородной СЛАУ \(Ax = 0\), где \(n\) – число неизвестных, \(r = \text{Rg}\, A\), называют фундаментальной системой решений (ФСР). Теорема (о существовании ФСР). Рассмотрим однородную СЛАУ \(Ax = 0\). В ней существует \(k = n - r\) линейно независимых решений, где \(n\) – число неизвестных, а \(r = \text{Rg}\, A\). Доказательство. Будем предполагать, что базисный минор расположен в левом верхнем углу. Тогда строки \(a_1,...,a_r\) являются базисными, а строки \(a_{r+1},...,a_m\) – их линейными комбинациями. После элементарных преобразований, обнуляющих зависимые строки, СЛАУ \(Ax = 0\) эквивалентна системе (⇓) с r уравнениями, где \(x_1,...,x_r\) – главные (базисные) переменные, а \(x_{r+1},...,x_n\) – свободные. Придаём свободным переменным поочерёдно наборы: 1-й набор: \(x_{r+1}=1, x_{r+2}=0,...,x_n=0\); 2-й набор: \(x_{r+1}=0, x_{r+2}=1,...,x_n=0\); ...; \((n-r)\)-й набор: \(x_{r+1}=0,...,x_n=1\). Для каждого набора свободных переменных решаем систему относительно \(x_1,...,x_r\). Получаем решения \(\xi_1,...,\xi_k\). Покажем, что они л.н.з. Рассмотрим равенство \(\varsigma_1\xi_1 + \cdots + \varsigma_k\xi_k = 0\). В нижних \(n-r\) координатах это даёт \(\varsigma_1 = \varsigma_2 = \cdots = \varsigma_k = 0\). Значит \(\xi_1,...,\xi_k\) – л.н.з. Их \(n-r\), и они являются решениями \(\Rightarrow\) это ФСР.
Сформулировать критерий существования ненулевого решения однородной системы линейных уравнений с квадратной матрицей и доказать его.	Утверждение. Однородная СЛАУ \(Ax = 0\) имеет ненулевые решения \(\Leftrightarrow \det A = 0\), т.е. A – вырожденная матрица. Доказательство. <br> Необходимость. Дано: \(Ax = 0\) имеет решение \(x \neq 0\). Доказать: \(\det A = 0\). Предположим, что \(\det A \neq 0 \Rightarrow\) СЛАУ имеет единственное решение (по правилу Крамера) и это решение \(x = 0\) – противоречие \(\Rightarrow \det A = 0\). <br> Достаточность. Дано: \(\det A = 0\). Доказать: \(\exists x \neq 0 : Ax = 0\). Определитель \(\det A = 0 \Rightarrow \text{Rg}\, A < n\). Пусть \(\text{Rg}\, A = r\). По теореме о существовании ФСР, существуют \(n-r > 0\) л.н.з. (ненулевых) решений СЛАУ \(Ax = 0\). Это и есть ненулевые решения.
Доказать теорему о структуре общего решения однородной системы линейных алгебраических уравнений.	Теорема (о структуре общего решения однородной СЛАУ). Пусть \(\xi_1,...,\xi_k\) – ФСР однородной СЛАУ \(Ax = 0\). Тогда любое решение этой СЛАУ можно представить в виде: \(x = c_1\xi_1 + \cdots + c_k\xi_k\), где \(c_1,...,c_k\) – некоторые постоянные. Доказательство. Пусть \(x^0\) – произвольное решение однородной СЛАУ \(Ax = 0\). Предположим, что базисный минор расположен в левом верхнем углу. После элементарных преобразований СЛАУ \(Ax = 0\) эквивалентна системе (1), из которой главные переменные выражаются через свободные (система (2)). Составим матрицу D из столбцов \(x^0, \xi_1,...,\xi_k\). Покажем, что \(\text{Rg}\, D = k\): 1) \(\text{Rg}\, D \geq k\), т.к. \(\xi_1,...,\xi_k\) – л.н.з. (по определению ФСР), а по теореме о ранге матрицы \(\text{Rg}\, D\) равен максимальному числу л.н.з. столбцов. 2) Покажем, что \(\text{Rg}\, D \leq k\). Столбцы \(x^0, \xi_1,...,\xi_k\) – решения СЛАУ \(Ax = 0\). Тогда для них верна система (2). После подстановки столбцов \(x^0, \xi_1,...,\xi_k\) на место x в системе (2), первая строка \(d_i\) матрицы D является линейной комбинацией строк \(d_{r+1},...,d_n\). Проводя элементарные преобразования, обнуляем первые r строк матрицы D. Получаем матрицу \(D_1\), у которой первые r строк нулевые, и \(\text{Rg}\, D_1 \leq n-r = k\). При элементарных преобразованиях ранг не меняется \(\Rightarrow \text{Rg}\, D \leq k\). Таким образом, \(\text{Rg}\, D = k\) (т.к. \(\text{Rg}\, D \geq k\) и \(\text{Rg}\, D \leq k\)). Заметим, что \(\xi_1,...,\xi_k\) являются базисными (они л.н.з. и их k = Rg D) \(\Rightarrow\) по теореме о базисном миноре столбец \(x^0\) – их линейная комбинация, т.е. \(\exists c_1,...,c_k : x^0 = c_1\xi_1 + \cdots + c_k\xi_k\).
